Программирование на Python. База.

Этот урок предназначен для абсолютных новичков в программировании. Мы познакомимся с базовыми концепциями Python и напишем нашу первую программу.

Что такое Python?

Python — это высокоуровневый язык программирования, известный своей простотой и читаемостью. Он используется во множестве областей, от веб-разработки до анализа данных и машинного обучения. Его синтаксис очень похож на обычный английский язык, что делает его относительно легким для освоения.

1) Установка Python:

Перед началом вам нужно установить Python на ваш компьютер. Скачайте последнюю версию с официального сайта python.org python.org и следуйте инструкциям по установке. Убедитесь, что вы добавили Python в переменные окружения вашей системы (это позволит запускать Python из командной строки).

2) Запуск Python:

После установки вы можете запустить Python двумя способами:

1. Интерактивный режим: Откройте командную строку (или терминал) и введите `python` (или `python3`, в зависимости от вашей системы). Вы увидите приглашение `>>>`, означающее, что вы находитесь в интерактивном режиме. Здесь вы можете писать код построчно и сразу видеть результаты.

2. Файл скрипта: Создайте текстовый файл (например, `my_program.py`) и напишите в нем код. Затем запустите его из командной строки, набрав `python my_program.py`.

Комментарии в коде — это текст, который игнорируется интерпретатором Python. Они используются для пояснения кода и повышения его читаемости. Комментарии начинаются с символа `#`
